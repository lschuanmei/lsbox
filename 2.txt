题目：git操作实战
1.安装
（1）下载
（2）安装
（3）查看版本号
>git --version

2.建立本地仓库
（1）创建目录
>mkdir /e/myweb
（2）初始化
>git init
（3）创建文件
（4）配置管理者
>git config --global user.email"1607481771@qq.com"
>git config --global user.name "lschuanmei"
(5)添加文件到本地仓库
>git add 文件名
>git commit -m "说明内容"
（6）查看提交情况
>git status
>git log
3.建立github仓库
（1）创建并添加密钥
本地创建
>ssh-keygen
将.ssh复制到仓库文件夹中
远程添加到SSH(id_rsa.pub内容)
（2）建立远程连接
>git remote add xxxx git@github.com：lschuanmei/myweb.git
（3）将本地内容上传到远程仓库中
>git remote show xxxx 
（4）将本地内容上传到远程仓库中
>git push xxxx master
